# Indian Agro Mart Frontend

A modern responsive frontend website for **Indian Agro Mart**, an agriculture machinery business website built to showcase products, collect customer enquiries, and improve online presence.

## Live Website

https://indian-agro-mart-frontend.vercel.app

## Project Overview

Indian Agro Mart is a business website designed for agriculture machinery and agro solutions.  
The goal of this project is to help customers view products, understand services, and submit enquiries easily.

## Features

- Modern responsive landing page
- Agriculture machinery product showcase
- Customer enquiry section
- Contact and business information section
- Mobile-friendly design
- Clean UI with agriculture theme
- Frontend deployed on Vercel
- Connected with backend API

## Tech Stack
